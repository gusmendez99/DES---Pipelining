# DES

## Description

DES algorithm implementation with pipelining, pthreads and conditional variables.

### Executing program

* How to run the program
```
$ ./g++ des-pipeline.cpp -o des -lpthread
$ ./des -d crypted.cry decrypt.txt keys.txt
```
